# Документация системы учета заявок на ремонт климатического оборудования

## Обзор созданных документов

В рамках выполнения задания 1 были созданы следующие документы:

### 1. algorithms_and_flowcharts.md
Основной документ с алгоритмами и блок-схемами системы:
- Основной алгоритм учета заявок
- Алгоритм расчета статистики  
- Алгоритм аутентификации пользователей
- Алгоритм назначения заявок на мастерские
- Алгоритм генерации QR-кодов
- Псевдокод для каждого алгоритма

### 2. module_specifications.md
Детальные спецификации модулей системы:
- Модуль статистики (StatisticsModule)
- Модуль отчетности (ReportsModule)
- Модуль управления заявками (RequestModule)
- Модуль аутентификации (AuthModule)
- Входные и выходные данные для каждого модуля
- API интерфейсы и коды ошибок

### 3. system_flowcharts.md
Визуальные блок-схемы в формате Mermaid:
- Основная блок-схема системы
- Блок-схема расчета статистики
- Блок-схема обработки заявок
- Блок-схема аутентификации
- Блок-схема генерации отчетов

## Соответствие требованиям

Все документы созданы в соответствии с:
- ГОСТ 19.701-90 "Схемы алгоритмов, программ, данных и систем"
- Требованиями технического задания
- Стандартами разработки ПО

## Использование документов

Эти документы служат основой для:
- Понимания архитектуры системы
- Разработки программного кода
- Тестирования функциональности
- Документирования системы
