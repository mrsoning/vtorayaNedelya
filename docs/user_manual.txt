РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ
СИСТЕМА УЧЕТА ЗАЯВОК НА РЕМОНТ КЛИМАТИЧЕСКОГО ОБОРУДОВАНИЯ

1. ВВЕДЕНИЕ

1.1 Назначение системы
Система предназначена для автоматизации процесса приема, обработки и контроля заявок на ремонт климатического оборудования. Система обеспечивает эффективное взаимодействие между заказчиками, специалистами и менеджерами сервисного центра.

1.2 Для кого предназначено руководство
Данное руководство предназначено для всех пользователей системы:
- Заказчики (клиенты сервисного центра)
- Операторы (принимающие заявки)
- Специалисты (выполняющие ремонт)
- Менеджеры (контролирующие процесс)
- Менеджеры по качеству (контроль качества работ)

2. НАЧАЛО РАБОТЫ

2.1 Вход в систему
1. Откройте веб-браузер (рекомендуется Chrome, Firefox, Edge)
2. Введите адрес системы: http://localhost:3001
3. На странице входа введите:
   - Логин (предоставляется администратором)
   - Пароль (предоставляется администратором)
4. Нажмите кнопку "Войти"

2.2 Первый вход
При первом входе рекомендуется:
- Ознакомиться с интерфейсом системы
- Проверить свои персональные данные
- Изменить пароль на более безопасный (если доступно)

2.3 Выход из системы
Для выхода из системы:
1. Нажмите на свое имя в правом верхнем углу
2. Выберите "Выход"
3. Вы будете перенаправлены на страницу входа

3. РАБОТА ДЛЯ ЗАКАЗЧИКОВ

3.1 Создание новой заявки
1. После входа нажмите "Мои заявки" в меню
2. Нажмите кнопку "Создать заявку"
3. Заполните форму:
   - Выберите тип оборудования (кондиционер, вентилятор и т.д.)
   - Выберите модель оборудования
   - Опишите проблему подробно
   - Укажите приоритет (обычный, срочный, очень срочный)
4. Нажмите "Создать заявку"
5. Система автоматически присвоит номер заявке

3.2 Просмотр своих заявок
1. Перейдите в раздел "Мои заявки"
2. Вы увидите список всех ваших заявок с информацией:
   - Номер заявки
   - Дата создания
   - Тип оборудования
   - Текущий статус
   - Назначенный специалист (если есть)
3. Нажмите на заявку для просмотра подробной информации

3.3 Отслеживание статуса заявки
Заявка может находиться в следующих статусах:
- "Новая заявка" - заявка создана, ожидает обработки
- "В процессе ремонта" - специалист работает над заявкой
- "Ожидание комплектующих" - ремонт приостановлен, ожидаются запчасти
- "Готова к выдаче" - ремонт завершен, можно забирать оборудование
- "Завершена" - заявка полностью закрыта
- "Отменена" - заявка отменена

3.4 Добавление комментариев
1. Откройте заявку
2. Прокрутите вниз до раздела "Комментарии"
3. Введите текст комментария
4. Нажмите "Добавить комментарий"
5. Комментарий будет виден всем участникам процесса

3.5 Оценка качества работ
После завершения ремонта:
1. Вы получите QR-код (на email или при выдаче оборудования)
2. Отсканируйте QR-код смартфоном
3. Откроется форма оценки качества
4. Выберите оценку:
   - "Хорошо" - работа выполнена отлично
   - "Нормально" - работа выполнена удовлетворительно
   - "Плохо" - работа выполнена некачественно
5. Добавьте комментарий (опционально)
6. Нажмите "Отправить оценку"

4. РАБОТА ДЛЯ ОПЕРАТОРОВ

4.1 Просмотр всех заявок
1. Перейдите в раздел "Все заявки"
2. Вы увидите список всех заявок в системе
3. Используйте фильтры для поиска нужных заявок:
   - По статусу
   - По типу оборудования
   - По дате создания
   - По приоритету

4.2 Обработка новых заявок
1. Откройте заявку со статусом "Новая заявка"
2. Проверьте описание проблемы
3. Назначьте специалиста:
   - Выберите специалиста из списка
   - Учитывайте загрузку специалистов
   - Учитывайте специализацию
4. Измените статус на "В процессе ремонта"
5. Сохраните изменения

4.3 Изменение статуса заявки
1. Откройте заявку
2. Выберите новый статус из выпадающего списка
3. При необходимости добавьте комментарий
4. Нажмите "Сохранить"

4.4 Назначение и переназначение специалистов
1. Откройте заявку
2. В поле "Специалист" выберите нужного специалиста
3. Система автоматически уведомит специалиста
4. Сохраните изменения

5. РАБОТА ДЛЯ СПЕЦИАЛИСТОВ

5.1 Просмотр назначенных работ
1. Перейдите в раздел "Мои работы"
2. Вы увидите список заявок, назначенных на вас
3. Заявки отсортированы по приоритету и дате

5.2 Начало работы над заявкой
1. Откройте заявку
2. Ознакомьтесь с описанием проблемы
3. Проверьте наличие необходимых комплектующих
4. Если все готово - начинайте работу
5. Если нужны запчасти - измените статус на "Ожидание комплектующих"

5.3 Обновление статуса работ
1. Откройте заявку
2. Выберите соответствующий статус:
   - "В процессе ремонта" - работа идет
   - "Ожидание комплектующих" - нужны запчасти
   - "Готова к выдаче" - ремонт завершен
3. Добавьте комментарий о проделанной работе
4. Сохраните изменения

5.4 Добавление информации о выполненных работах
1. Откройте заявку
2. В разделе "Комментарии" добавьте информацию:
   - Что было сделано
   - Какие запчасти использованы
   - Сколько времени заняла работа
   - Рекомендации для клиента
3. Нажмите "Добавить комментарий"

5.5 Завершение работы
1. После завершения ремонта измените статус на "Готова к выдаче"
2. Добавьте итоговый комментарий
3. Система автоматически уведомит клиента
4. Система сгенерирует QR-код для оценки качества

6. РАБОТА ДЛЯ МЕНЕДЖЕРОВ

6.1 Мониторинг всех заявок
1. Перейдите в раздел "Все заявки"
2. Используйте фильтры для анализа:
   - Заявки по статусам
   - Заявки по специалистам
   - Просроченные заявки
   - Заявки по приоритету

6.2 Просмотр отчетов и статистики
1. Перейдите в раздел "Отчеты"
2. Доступные отчеты:
   - Общая статистика по заявкам
   - Статистика по типам оборудования
   - Статистика по статусам заявок
   - Производительность специалистов
   - Среднее время выполнения работ

6.3 Экспорт отчетов в PDF
1. Откройте нужный отчет
2. Нажмите кнопку "Экспорт в PDF"
3. Система сгенерирует PDF файл
4. Файл автоматически загрузится на ваш компьютер
5. PDF содержит:
   - Все данные из веб-версии отчета
   - Дату и время генерации
   - Графики и таблицы

6.4 Управление процессом
1. Контролируйте загрузку специалистов
2. Перераспределяйте заявки при необходимости
3. Отслеживайте просроченные заявки
4. Анализируйте причины задержек
5. Принимайте управленческие решения на основе статистики

6.5 Работа с проблемными заявками
1. Выявляйте заявки с длительным временем выполнения
2. Связывайтесь со специалистами для выяснения причин
3. При необходимости переназначайте заявки
4. Информируйте клиентов о задержках
5. Документируйте причины задержек в комментариях

7. РАБОТА ДЛЯ МЕНЕДЖЕРОВ ПО КАЧЕСТВУ

7.1 Контроль качества работ
1. Просматривайте завершенные заявки
2. Анализируйте оценки клиентов
3. Выявляйте проблемные случаи
4. Связывайтесь с клиентами для уточнения

7.2 Консультации по сложным случаям
1. Специалисты могут обращаться к вам за консультацией
2. Просматривайте заявки, требующие экспертизы
3. Добавляйте рекомендации в комментариях
4. При необходимости участвуйте в решении проблемы

7.3 Продление сроков выполнения
1. Откройте заявку, требующую продления срока
2. Оцените обоснованность запроса
3. Измените плановую дату завершения
4. Добавьте комментарий с обоснованием
5. Уведомите клиента о продлении срока

7.4 Анализ качества обслуживания
1. Просматривайте статистику оценок
2. Выявляйте тренды в качестве работ
3. Анализируйте причины низких оценок
4. Разрабатывайте рекомендации по улучшению
5. Проводите обучение специалистов

8. ОБЩИЕ ФУНКЦИИ

8.1 Поиск заявок
1. Используйте поле поиска в верхней части страницы
2. Можно искать по:
   - Номеру заявки
   - Описанию проблемы
   - Имени клиента
   - Типу оборудования
3. Результаты отображаются в реальном времени

8.2 Фильтрация данных
1. Используйте фильтры на странице списка заявок
2. Доступные фильтры:
   - По статусу
   - По типу оборудования
   - По дате создания
   - По приоритету
   - По специалисту
3. Можно комбинировать несколько фильтров

8.3 Сортировка данных
1. Нажмите на заголовок столбца для сортировки
2. Первое нажатие - сортировка по возрастанию
3. Второе нажатие - сортировка по убыванию
4. Третье нажатие - отмена сортировки

8.4 Уведомления
Система автоматически отправляет уведомления:
- При создании новой заявки
- При назначении специалиста
- При изменении статуса заявки
- При добавлении комментария
- При завершении работ

9. ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ

9.1 Как изменить пароль?
Обратитесь к администратору системы для смены пароля.

9.2 Что делать, если забыл пароль?
Обратитесь к администратору системы для восстановления доступа.

9.3 Как отменить заявку?
Свяжитесь с оператором или менеджером для отмены заявки.

9.4 Почему я не вижу некоторые заявки?
Доступ к заявкам зависит от вашей роли в системе. Заказчики видят только свои заявки, специалисты - назначенные на них, менеджеры - все заявки.

9.5 Как долго хранятся данные в системе?
Все данные хранятся бессрочно. Архивные заявки можно просмотреть через фильтры.

9.6 Можно ли прикрепить файлы к заявке?
В текущей версии системы прикрепление файлов не поддерживается. Используйте подробное текстовое описание.

9.7 Как связаться с технической поддержкой?
Обратитесь к администратору системы или в IT-отдел вашей организации.

10. СОВЕТЫ ПО ЭФФЕКТИВНОЙ РАБОТЕ

10.1 Для заказчиков
- Описывайте проблему максимально подробно
- Указывайте точную модель оборудования
- Проверяйте статус заявки регулярно
- Оставляйте оценку качества после завершения работ

10.2 Для операторов
- Обрабатывайте новые заявки оперативно
- Назначайте специалистов с учетом их загрузки
- Добавляйте комментарии при изменении статуса
- Информируйте клиентов о ходе работ

10.3 Для специалистов
- Обновляйте статус заявки своевременно
- Добавляйте подробные комментарии о работе
- Сообщайте о проблемах заранее
- Завершайте работы в установленные сроки

10.4 Для менеджеров
- Проверяйте отчеты ежедневно
- Анализируйте тренды и закономерности
- Оперативно реагируйте на проблемы
- Используйте данные для принятия решений

11. ЗАКЛЮЧЕНИЕ

Система учета заявок на ремонт климатического оборудования разработана для повышения эффективности работы сервисного центра. Следуя инструкциям данного руководства, вы сможете максимально эффективно использовать все возможности системы.

При возникновении вопросов или проблем обращайтесь к администратору системы или в службу технической поддержки.

Желаем вам продуктивной работы!
