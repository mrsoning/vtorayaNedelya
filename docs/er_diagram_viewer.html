Инструкции по экспорту в PNG

Для создания PNG версии диаграммы:

  1. Откройте этот файл в браузере
  2. Нажмите правой кнопкой мыши на диаграмме
  3. Выберите "Сохранить изображение как..." или используйте инструменты разработчика для экспорта SVG
  4. Или используйте функцию печати браузера для сохранения в PDF

  • docs/er_diagram_viewer.html - Этот HTML файл для просмотра

Описание диаграммы

Данная ER-диаграмма представляет полную структуру базы данных системы управления заявками на ремонт климатического оборудования в третьей нормальной форме (3НФ) с обеспечением ссылочной целостности.

Основные характеристики:

  • 8 основных сущностей с правильно определенными атрибутами
  • Первичные ключи (PK) для каждой таблицы
  • Внешние ключи (FK) для обеспечения ссылочной целостности
  • Уникальные ключи (UK) для бизнес-логики
  • Правильные кардинальности связей (1:1, 1:M)
  • Соответствие 3НФ - отсутствие избыточности и транзитивных зависимостей
